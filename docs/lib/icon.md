# Icon

Component wrapper for @mdi/js.

## Usage

```svelte
<script lang="ts">
	import { mdiAccount } from '@mdi/js';
</script>

<Icon icon="mdiAccount" />
```

## Props

| **Prop** | **Type** | **Description**          | **Required** |
| -------- | -------- | ------------------------ | ------------ |
| Icon     | String   | Path string from @mdi/js | True         |
| Size     | String   | Prest sizes              | -            |
